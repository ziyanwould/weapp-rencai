# 微信小程序-拉勾


### 说明：

简单的拉勾客户端，主要实现了职位展示页面和底部4个页面切换功能。

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目底部tab图片
- pages — 存放项目页面相关文件，一共包括detail,index,logs,user等几个页面
- utils — 存放日期格式化文件

### 开发环境：

微信web开发者工具 v0.10.102800
